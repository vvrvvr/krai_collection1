Привет, кратко по классам:

InstantiateLevel - создает пул обьектов 
(в данном примере их два (бэкграунд и интерактивные)). 
После чего контролирует какие и когда включать.
так же двигает объекты на уровне.

ImageSwap - меняет картинку вагона на зачеркнутую.

VideoController - контролирует вклчюение стартового видео. 
Если игру только запустили, видео показывается, 
если игрок вышел в главное меню из основной игры, 
то видео не показывается повторно. 
Так же можно пропустить видео нажав Space 

MainMenu - класс контроллер (вкл/пауза/выкл) 
(ВАЖНО, использована DontDestroyOnLoad)

RedIO, BlueIO - классы для смены на аноним, 
и для смены на перевод (Рандом из двух)

WhiteIO - класс записи в строку игрока фразы 
записанной в строке объекта.

ReturnInPoolController - когда объект на 
уровне выходит за указанную дистанцию, 
отправляем его обратно в пул

GameController - контроль игры и данных

HandController - контроль руки, ее атаки и положения
Так же выполняет роль своеобразного саунд менеджера.

AssetText - набор фраз. 
В общем: у нас есть 30 Предложений, 
каждое по 3 слова. И у каждого слова есть перевод и аноним. 
Когда нам нужно заменить конкретное слово, 
мы можем легко обратиться в набор слов 
конкретно по этому слову и заменить на нужное

RunWriteController - фактически PlayerController. 
Управление персонажем, а так же вагонами идущими за ним
